# ПРОЕКТ: Сайт репетитора финского языка
# ТЕХНОЛОГИИ: Astro + React/TypeScript для интерактивных компонентов + CSS Modules
# ЦЕЛИ:
o  	На первом этапе: привлекать новых учеников взрослых, желающих выучить финский для работы, а также привлекать родителей русскоязычных учеников, живущих в Финляндии, чьим детям нужен репетитор, чтобы лучше успевать в школе
o	На втором этапе: записать уроки, возможно сделать интерактивные курсы и продавать их.
o	Целевая аудитория: взрослые, студенты, школьники, изучающие финский язык. Образовательная платформа с упором на производительность и SEO

## ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ:
- Фреймворк: Astro (статическая генерация)
- Стили: CSS Modules + кастомные переменные
- Интерактивность: React + TypeScript только где необходимо
- Хостинг: Github
- Все размеры в rem.
- Используем типографику из global.css
